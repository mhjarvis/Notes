# General
Semantic HTML helps bring meaning and context to HTML. There are seven native HTML elements that define landmark regions:

* ```<aside>```
* ```<footer>```
* ```<form>```
* ```<header>```
* ```<main>```
* ```<nav>```
* ```<section>```
